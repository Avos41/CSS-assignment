CSS Positioning Techniques

Static Positioning

This element is positioned according to the normal flow of the document. It is the default positioning for any HTML element.

Relative Positioning

This element is positioned relative to its normal position. The position is adjusted by 10px from the top and 20px from the left.

Absolute Positioning

This element is positioned absolutely to its nearest positioned ancestor. It is 50px from the top and 20px from the right of the container.

Fixed Positioning

This element is positioned relative to the viewport. It stays in the same place even if the page is scrolled.

Sticky Positioning

This element is positioned based on the user's scroll position. It toggles between relative and fixed positioning, depending on the scroll position. Scroll down to see it in action.
